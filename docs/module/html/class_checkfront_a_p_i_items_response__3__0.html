isError ( )

Check if the request was made succesfulyl and responded with something.

If this returns true then getNativeCode should return the API result code if there is one, and getMessage should return the error message if there is one.

NB for HTTP errors an exception will be thrown instead as that's bad news, maybe config but probably not recoverable.

Returns
boolean true if request failed (bad url, invalid parameters passed etc) false if something returned (maybe empty though)

Implements CheckfrontAPIResponseInterface.
